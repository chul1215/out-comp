# 유성선병원 2026년 연간 홍보 계획 및 전략

> **작성일**: 2026-03-10
> **기준 데이터**: 2025년 1월~12월 외래환자 진료실적 (D25 내장 데이터)
> **적용 기간**: 2026년 1월 ~ 12월 (연간)

---

## 1. 문서 개요

### 1-1. 목적

2025년 외래환자 데이터 분석 결과를 기반으로, 유성선병원의 2026년 연간 홍보 전략과 월별 실행 계획을 수립한다. 데이터에서 도출된 5대 타겟 진료과(정형외과, 소화기내과, 이비인후과, 심장내과, 신경과)를 중심으로, 계절별 환자 수요 패턴에 맞춘 체계적 마케팅 실행 로드맵을 제시한다.

### 1-2. 전략 방향성

| 방향 | 핵심 내용 |
|------|----------|
| **거점 진료과 브랜드 강화** | 심장내과(9,354명)·신경과(9,116명) 초대형 진료과의 지역 1위 포지션 공고화 |
| **성장 동력 가속** | 정형외과(+8.1%), 관절센터(+24.0%) 등 성장 진료과 마케팅 집중 투자 |
| **건강검진 크로스셀링** | 건강검진 → 심혈관·소화기 정밀검사 전환 파이프라인 구축 |
| **지역 밀착 네트워크** | 유성구 테크노단지 기업 단체검진, 시니어 복지관 연계 |

### 1-3. 적용 기간

2026년 1월 1일 ~ 12월 31일 (12개월)

### 2-1. 2025년 외래 실적 포지셔닝

### 2-2. 대전선 대비 차별화 포인트

**유성선만의 강점 4가지:**

1. **심혈관 거점 병원**: 심장내과 9,354명으로 대전선(5,448명) 대비 1.7배. 건강검진(Z000)이 1위 진단으로, 검진 연계 심혈관 스크리닝의 최적 기반 보유
2. **대형 정형외과 수술 거점**: 정형외과 6,271명(대전선 2.6배). 무릎관절 수술, 회전근개, 골절 등 대형 수술 역량
3. **뇌건강 포괄 관리 허브**: 신경과 9,116명(대전선 1.6배). 동맥경화 + 뇌경색 + 경도 인지장애(치매 전단계)까지 원스톱 관리
4. **검진형 소화기센터**: 복통·대장 폴립 중심의 검진 수요가 높아 건강검진 연계 효과 극대화 가능

### 2-3. 유성구 지역 특성

- **유성구 인구**: 약 37만 명, 대전 5개 구 중 최대 인구 규모
- **신도심 밀집**: 봉명동·도안신도시·전민동 등 30~50대 가족 단위 거주 밀집
- **기업 밀집**: 대덕테크노밸리, 대덕연구단지, 유성구 산업단지 등 연구·기술직 종사자 다수
- **보험 비율 94.4%**: 안정적 소득 기반의 건강보험 가입자 중심 → 검진·예방 마케팅 반응도 높음

## 3. 연간 홍보 전략 방향 — 4대 전략축

### 전략축 1: 브랜드 포지셔닝 강화

**목표**: "유성 지역 심혈관·뇌건강·정형외과의 대표 병원" 인식 확립

| 실행 항목 | 세부 내용 |
|----------|----------|
| 브랜드 슬로건 | "유성선, 당신의 건강을 지키는 가장 가까운 종합병원" |
| 거점 진료과 브랜딩 | 심장내과·신경과·정형외과 3대 센터 브랜드 개별 구축 |
| 전문의 프로필 홍보 | 각 센터장 인터뷰, 전문 분야 소개 콘텐츠 제작 |
| 지역 미디어 노출 | 대전 지역 언론 건강 칼럼 기고 (월 2회), 지역 방송 건강 코너 출연 |

### 전략축 2: 신환 유입 확대

**목표**: 타겟 5개 진료과 신환 유입률 전년 대비 15% 향상

| 실행 항목 | 세부 내용 |
|----------|----------|
| 검색광고 최적화 | 진료과별 핵심 키워드 네이버/구글 검색광고 운영 (증상 기반 키워드 중심) |
| SNS 증상 콘텐츠 | 인스타그램·블로그에 증상별 자가체크 콘텐츠 발행 (주 3회) |
| 지역 커뮤니티 침투 | 유성구 맘카페, 아파트 커뮤니티, 직장인 커뮤니티 정보성 게시 |
| 기업 단체검진 영업 | 유성구 테크노단지·연구단지 기업 대상 단체검진 제안 (상반기 40개사 목표) |

### 전략축 3: 재내원율 제고

**목표**: 타겟 진료과 재내원율 5%p 향상 (특히 IMC·NR의 만성질환 관리군)

| 실행 항목 | 세부 내용 |
|----------|----------|
| 카카오톡 알림 시스템 | 진료 후 추적검사 일정 리마인드, 복약 안내 메시지 |
| 건강관리 콘텐츠 발송 | 만성질환 환자 대상 월 1회 맞춤형 건강정보 카카오 발송 |
| 정기검진 프로그램 | 심혈관(연 2회)·뇌혈관(연 1회) 정기 추적검사 패키지 |
| 진료 만족도 관리 | 진료 후 24시간 내 만족도 설문 → 불만 사항 48시간 내 응대 |

### 전략축 4: 계절 특화 캠페인

**목표**: 계절별 환자 수요 변동에 선제적으로 대응하여 비수기 환자 감소 최소화

| 계절 | 핵심 캠페인 | 주력 진료과 |
|------|-----------|------------|
| 봄 (3~5월) | 야외활동 시작 + 건강검진 시즌 | OS, IMG, ENT, IMC, NR |
| 여름 (6~8월) | 여름 건강관리 + 수면 질환 | ENT, IMG, NR |
| 가을 (9~11월) | 환절기 심뇌혈관 + 관절 관리 | IMC, NR, OS, JC |
| 겨울 (12~2월) | 겨울철 뇌졸중 예방 + 신년 검진 | NR, IMC, IMG |

| 지역 언론 | 심혈관질환 예방 관련 건강 칼럼 기고 (계절별 맞춤) | 월 1~2회 |

**프로모션 계획**

- 봄(3~5월): 심혈관 정밀검사 패키지 (심장초음파 + 경동맥초음파 + 24시간 심전도)
- 가을(9~11월): 환절기 심혈관 점검 캠페인, 기업 단체검진 하반기 추가 영업
- 겨울(12~2월): 겨울철 돌연사 예방 캠페인, 고혈압 관리 강화 콘텐츠

**크로스셀링 전략 (핵심)**

```
건강검진 수검 → 이상 소견 발견 (혈압·심전도·혈액검사)
                    ↓
              심장내과 정밀검사 안내 (심장초음파, 관상동맥 CT)
                    ↓
              고위험군 → 정기 추적관리 프로그램 등록
              저위험군 → 1년 후 재검진 리마인드
```

### 4-5. 신경과 (NR) — 뇌건강 포괄 관리 허브

| 항목 | 세부 내용 |

| **핵심 메시지** | "뇌건강, 예방이 최선입니다. 유성선 신경과, 뇌혈관 정밀검진부터 치매 예방까지" |

| 채널 | 활용 방안 | 집행 시기 |
|------|----------|----------|

| 네이버 블로그 | 두통 유형별 원인 분석, 뇌경색 초기 증상 FAST 안내, 치매 예방법 | 주 2회 |
| 카카오톡 | 뇌혈관질환 관리 환자 추적검사 리마인드 | 월 2회 |
| 시니어 복지관 | 유성구 노인복지관·경로당 대상 뇌건강 무료 강좌 | 분기 1회 |
| 심장내과 연계 | 심혈관 환자 대상 뇌혈관 동시 검진 안내 (동맥경화 공통 기반) | 연중 |

**프로모션 계획**

- 봄(3~5월): 환절기 뇌혈관 정밀검사 (MRA + 경동맥초음파) 패키지
- 가을(9~11월): 치매 조기검진 캠페인 (9월 21일 세계 알츠하이머의 날 연계)
- 겨울(12~2월): 겨울철 뇌졸중 예방 캠페인 (한랭 자극에 의한 혈관 수축 경고)

**IMC-NR 연계 전략**

심장내과와 신경과는 동맥경화(I70990)를 공통 기반으로 환자 교차 의뢰가 가능하다.

```
심장내과 환자 (고혈압·동맥경화) → 뇌혈관 검사 동시 권고
신경과 환자 (동맥경화·뇌경색) → 심혈관 검사 동시 권고
→ "심뇌혈관 동시 정밀검진 패키지" 상품화
```

## 5. 계절별 캠페인 계획

### 5-1. 봄 캠페인 (3~5월) — 연간 최대 마케팅 시즌

**캠페인명**: "봄, 건강을 점검하는 계절"

| 구분 | 내용 |
|------|------|
| **기간** | 2월 말 사전 홍보 개시, 3월 본격 론칭, 4월 피크, 5월 마무리 |
| **예산 배분** | 연간 홍보 예산의 **35%** 집중 투입 |
| **목표** | 봄 시즌 신환 전년 대비 +15%, 건강검진 연계 정밀검사 전환율 20% |

**캠페인 A: 봄 건강검진 시즌 — "올 봄, 건강검진 제대로 받으세요"**

| 항목 | 내용 |
|------|------|
| 대상 | 30~60대 직장인, 유성구 기업체 근로자 |
| 주력 진료과 | IMG, IMC |
| 핵심 상품 | 위·대장 동시 내시경 패키지, 심혈관 정밀검진 추가 옵션 |
| 채널 | 기업 단체검진 영업 + 네이버 검색광고 + 카카오톡 |
| 메시지 | "건강검진, 기본에서 끝내지 마세요. 내시경 + 심혈관 정밀검진까지" |
| KPI | 기업 단체검진 계약 20개사, 내시경 검진 건수 전년 대비 +10% |

**캠페인 B: 봄 야외활동 시즌 — "운동 시작 전, 관절부터 점검하세요"**

| 항목 | 내용 |
|------|------|
| 대상 | 40~70대 야외활동 참여자, 스포츠 동호회 |
| 주력 진료과 | OS, JC |
| 핵심 상품 | 골밀도 검사 + 무릎 X-ray 패키지, 어깨 통증 전문 상담 |
| 채널 | 옥외광고 + 지역 동호회 + 네이버 검색광고 |
| 메시지 | "무릎·어깨, 아프면 참지 마세요. 봄 운동 전 관절 건강 체크" |
| KPI | OS 신환 전년 대비 +15%, 관절센터(JC) 신환 +20% |

**캠페인 C: 봄 알레르기 시즌 — "봄 코막힘, 비염인지 확인하세요"**

| 항목 | 내용 |
|------|------|
| 대상 | 전 연령대 알레르기 비염 환자, 소아 비염 보호자 |
| 주력 진료과 | ENT |
| 핵심 상품 | 알레르기 검사 패키지, 부비동염 치료 프로그램 |
| 채널 | 네이버 검색광고 + 인스타그램 + 맘카페 |
| 메시지 | "감기인 줄 알았는데 알레르기 비염? 정확한 검사로 원인을 찾으세요" |
| KPI | ENT 3~4월 신환 전년 대비 +15% |

**캠페인 D: 환절기 심뇌혈관 예방 — "환절기, 심장과 뇌를 지키세요"**

| 항목 | 내용 |
|------|------|
| 대상 | 50대 이상 만성질환자, 고혈압·당뇨 동반 환자 |
| 주력 진료과 | IMC, NR |
| 핵심 상품 | 심뇌혈관 동시 정밀검진 패키지 (심장초음파 + MRA + 경동맥초음파) |
| 채널 | 카카오톡 기존 환자 + 병원 내 안내 + 지역 언론 |
| 메시지 | "봄 환절기, 급격한 기온 변화에 혈관이 위험합니다" |
| KPI | 심뇌혈관 동시 검진 패키지 월 50건 이상 |

### 10-3. 성공의 핵심 요인

1. **건강검진 크로스셀링 파이프라인**: 건강검진 → IMC/IMG 정밀검사 전환이 유성선의 가장 큰 수익 기회. 검진센터 결과 상담 시 정밀검사 안내 프로세스를 표준화하는 것이 최우선 과제

2. **심뇌혈관 동시검진 패키지**: IMC(9,354명)와 NR(9,116명)이 동맥경화를 공유하는 점을 활용, 심뇌혈관 동시검진 패키지를 상품화하면 환자당 검사 단가와 만족도를 동시에 높일 수 있음

3. **기업 단체검진 선점**: 유성구 테크노단지·연구단지의 30~50대 건강보험 가입자(94.4%)는 검진형 마케팅에 최적 타겟. 상반기 조기 영업으로 경쟁 병원 대비 선점 필요

4. **데이터 기반 월별 조정**: 매월 신환 유입, 검색광고 전환율, 카카오톡 반응률 등 핵심 지표를 모니터링하고, 분기별로 예산·메시지·채널 믹스를 조정
