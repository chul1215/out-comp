### 1. 와이어프레임

> 모든 화면은 **병원 토글(대전선/유성선) + 기간 선택(월별/분기별/계절별)** 을 공통 헤더로 보유
> 6개 메인 뷰 + 데이터 업로드 화면으로 구성
>
> **v2.0 변경 사항 (2026-02-22)**
> - 프로그램 실행 시 25년 데이터 즉시 표시 (업로드 없이)
> - 데이터 업로드 화면: 26년도 파일만 업로드 (전년도 업로드 항목 제거)
> - YoY 비교: 25년(내장) vs 26년(업로드) 자동 구성
