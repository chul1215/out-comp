# 마케팅 활용 외래환자 비교 프로그램

> 2025년 외래환자 데이터 기반 마케팅 타겟 진료과/진단 분석 대시보드

## 개요

대전선병원 · 유성선병원의 2025년 외래환자 데이터를 분석하여 2026년 마케팅 타겟을 도출하는 단일 HTML 파일 대시보드입니다.

- **서버 불필요**: `dashboard.html` 더블클릭으로 바로 실행
- **25년 데이터 내장**: 파일 열면 즉시 분석 화면 표시
- **26년 데이터 업로드**: 업로드 후 YoY 비교 활성화
- **오프라인 완전 동작**: Chart.js + SheetJS 라이브러리 내장

## 사용 방법

1. `dashboard.html` 파일을 Chrome 또는 Edge 브라우저로 열기
2. 상단 **병원 토글**로 대전선/유성선 전환
3. **기간 선택**(월별/분기별/계절별)으로 분석 기간 변경
4. **26년 데이터 업로드** 버튼으로 26년 엑셀 파일 업로드 → YoY 비교 활성화

## 주요 기능

| 화면 | 기능 |
|------|------|
| 홈 | KPI 카드 · 진료과 TOP 10 · 월별 추이 |
| 월별트렌드 | 환자유형별 스택드바 · 보험유형별 라인차트 · 성장률 테이블 |
| 분기/계절 | 분기별/계절별 그룹드바 · 기간별 TOP 5 |
| 마케팅타겟 | 복합점수 가중치 슬라이더 · 진료과/진단 TOP 10 · 자동 인사이트 |
| YoY비교 | 증감 KPI · 진료과별/진단별 증감 테이블 *(26년 업로드 후 활성화)* |
| 진단상세 | 진단 TOP 20 · 월별 스파크라인 · 의사별 분포 |

## 제외 진료과 규칙

| # | 규칙 | 설명 |
|---|------|------|
| 1 | `DN`으로 시작 | 치과 계열 |
| 2 | `HPC` | 건강검진센터 |
| 3 | `FM` | 가정의학과 |
| 4 | 코드 끝에 `2` | 미수(수납 미완료) 코드 |

## 복합 점수 산출

```
Composite = 0.40 × Growth_Score + 0.30 × Volume_Score + 0.30 × NewPatient_Score
```

각 점수는 min-max 정규화 후 100점 만점으로 환산. 가중치는 슬라이더로 실시간 조절 가능.

## 기술 스택

- HTML5 + CSS3 + Vanilla JS (단일 파일)
- Chart.js 4.4.0 (인라인 내장)
- SheetJS 0.20.1 (인라인 내장)
